# Advent of Code

Practicing [Go](https://go.dev/) with [Advent of Code](https://adventofcode.com).

# Quickstart

- From any day folder, run the script with:

      cd 2025/1/part_one
      go run .

- Run tests:

      cd 2025/1/part_two
      go test

- Run specific test:

      cd 2025/1/part_two
      go test -run TestZeroCountL75
